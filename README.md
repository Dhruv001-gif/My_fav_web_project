# My Frequently Visited Websites 🌐

A simple and informative web directory highlighting my top three go-to platforms for learning and productivity. This project was built to practice HTML structure, media embedding, and hyperlinking.

## 🚀 Featured Platforms
*   **GeeksforGeeks:** My primary resource for Data Structures and Algorithms (DSA).
*   **Wikipedia:** The ultimate encyclopedia for deep-diving into new topics.
*   **ChatGPT:** My AI assistant for coding, creative writing, and problem-solving.

## 🛠️ Technical Implementation
*   **Image Links:** Used `<a>` tags around `<img>` to create clickable visual navigation.
*   **Text Formatting:** Utilized `<pre>` tags to maintain the layout of detailed descriptions.
*   **Responsive Basics:** Included meta viewport tags for better mobile viewing.
*   **Target Attributes:** Implemented `target="_blank"` to ensure external links open in a new tab, keeping the user on my page.

## 📖 How to Run
Simply open the `index.html` file in any modern web browser.
